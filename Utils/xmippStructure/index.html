Xmipp structureпѓЃ

XMIPP software is split in three repositories (XmippCore, XmippViz and Xmipp) plus the repository regarding to the ScipionвЂ™s plugin (scipion-em-xmipp). All this four repositories are in I2PC gitHub.

  • scipion-em-xmipp: It is the ScipionвЂ™s plugin. Protocol (wrappers), viewers, testsвЂ¦ are there. All Xmipp regarding Scipion is there.

  • xmipp: EM-reconstraction methods. Usually wrote in C++.

  • xmippCore: Low-level image-processing and EM-images definitions. In addition, metadata_labels, metadata and data are there.

  • xmippViz: Basically ShowJ, coordinates viewer and Xmipp Picker. Java code.
